# OCT Eye Image Anomaly Detection

## Overview
This project implements a deep learning model for detecting anomalies in Optical Coherence Tomography (OCT) retinal images. The model is trained to classify OCT scans and identify abnormalities in retinal structures.

## Dataset
The project uses the Kermany2018 OCT dataset, which contains retinal OCT images for training and validation.

**Dataset Source:** [Kermany2018 on Kaggle](https://www.kaggle.com/datasets/paultimothymooney/kermany2018)

## Project Structure
- `Retinal Anomaly Detection.ipynb` - Main Jupyter notebook containing the complete pipeline for data processing, model training, and evaluation

## Requirements
- Python 3.x
- TensorFlow/Keras
- NumPy
- Pandas
- Matplotlib
- kagglehub
- Other standard ML libraries

## Usage
1. Clone this repository
2. Download the Kermany2018 dataset from Kaggle
3. Open and run the `Retinal Anomaly Detection.ipynb` notebook
4. Follow the cells sequentially to train and evaluate the model

## Model
The notebook implements a deep learning model for OCT image classification and anomaly detection in retinal scans.
